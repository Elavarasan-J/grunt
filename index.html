Steps to clone the new repository

1. git init
2.git add .
3. git commit -m "Commit msg"
4. git remote add origin 'repo URL' eg:https://github.com/Elavarasan-J/myWorks.git
5. git push -u origin master



Steps to push the files after clone

1. git add .
2. git commit -m "commit msg"
3. git push -u origin master
